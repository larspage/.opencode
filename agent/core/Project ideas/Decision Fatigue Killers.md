## 🍔 Decision Fatigue Killers

These are your strongest “viral” tools.

-   **“Just Pick Dinner”**
    -   Input: location, cuisine vibes, price, dietary needs
    -   Output: _one_ place. No scrolling. No debate.
    -   Bonus: “Convince my group” button that generates a persuasive blurb
-   **“What Should I Eat This Week?”**
    -   Auto-generates a 5-day meal plan + grocery list
    -   Integrates laziness level (“I will cook” vs “microwave only”)
-   **“Weekend Decider”**
    -   You and friends vote on activities asynchronously
    -   Kills the “what do you want to do?” loop

## 🧠 Life Organization (Without Feeling Like Work)

-   **“Minimum Viable Day”**
    -   Forces you to pick only 3–5 _non-negotiable_ tasks
    -   Everything else is optional (anti-overwhelm design)
-   **“Life Buckets Tracker”**
    -   Tracks categories like Health, Money, Relationships, Home
    -   Shows which area you’re neglecting (simple visual dial)
-   **“The ‘I’ll Do It Later’ List”**
    -   A guilt-free dumping ground
    -   Periodically resurfaces items with: “Still care about this?”

## 💸 Money & Adult Responsibilities

-   **“Where Did My Money Go (Simple Edition)”**

## 💡 Strategic Advice (this matters more than the ideas)

A lot of these ideas already exist in some form—but they fail because they:

-   Ask for too much input
-   Try to do too much
-   Feel like work

If you want this to succeed:

**1\. Bias toward “instant answers”**

-   Users should get value in <10 seconds

**2\. Embrace constraints**

-   Example: only show ONE restaurant, not 20

**3\. Lean into tone**

-   Slightly sarcastic, empathetic, “we get it”

**4\. Build a shared design system**

-   Same UX across all tools → feels like a suite, not random sites

**5\. Start with 1–2 killer tools**

-   My picks:
    -   “Just Pick Dinner”
    -   “Minimum Viable Day”
